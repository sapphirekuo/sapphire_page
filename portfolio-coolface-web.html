CoolFace Website Project

The Project

CoolFace is a professional tent and camping equipment rental company. Their current online order system is separated into two part. One is using blog to display the product, and then user need to use another online registration system to reserve their order.

The goal of this web application is to integrate the process in one site.

User story

Tab Bar
  • User can use Tab Bar to switch between Home/ Products/Cart/ Orders/Profile Pages.
    • Login

  • User need to Login or Register before entering checkout page, orders page, and profile pages.
  • User can Login or register with email, FB and Google accounts.
  • User can click forget password button to send password to registered email.
    • Home
  • User can enter product page after choose departure (default: today + 7 days) and return (default: today + 9 days) dates on Home page.
  • User can click News links on Home page to browse in modal window.
    • Product
  • User can browse product lists on Product page.
  • User can click product list to browse product details in modal window.
  • User can add product into shopping cart. Successful massage will show after added.
  • User can slide product list left to show add-to -cart icon
  • User can edit rental dates in Product page. Successful massage will show after edited.
  • User can see rental price based on the days of rental.
  • User can click category icons to skip to certain category products.
    • Cart
  • User can edit rental dates in Cart page. Successful massage will show after edited. Priced will change accordingly.
  • User can change quantities in Cart page. Successful massage will show after edited. Priced will change accordingly.
  • User can click product list in the shopping cart to see details on Cart page.

  • User can place order after click Confirm button.
    • Order
  • User can brows order summery in Order page.
  • User can click single order to see details in modal window.
  • User can click Pay button in the details modal to third-party payment page.
    • Profile
  • User can edit personal info (excluding email) in Profile page
  • User can Logout in Profile page.
