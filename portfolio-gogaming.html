Gogoro Gamification Project

The Project

This team project is cooperate with Gogoro company to provide a gamification experience for the Gogoro bikersbikers to increase interactions with their Gogoro scooters. Therefore mileage will be increased and deep connection with Gogoro Brand will be bonded.

Goal

In this project, there are two main goal.

  • First, Gogoro wants to optimize go station ( battery exchange station) search.
  • Second, create an interaction game for biker to play with.
    • Survey

    After summing-up the survey from Gogoro bikers, the result shows most of the biker live in urban area and use Gogoro scooter to commute. Thus we created small missions of one day trip in the urban and suburban areas with theme to encourage Gogoro bikers to hang around during their time off.

    Gamification Elements

    Based on the existing Gogoro Badge system, a points reward system was designed for bikers to redeem Gogoro products. Points can be earned from go-stations (battery exchange system) and mission completion. Also, extra points will be award for badges with achievements.

    Social interactions are also included for more fun.

    Discussions

    #1: Navigation

    • Originally a hamburger menu icon was used to switch between functions. However, it’s not friendly to the bikers to use while moving.
    • There are two main independent functions - go station search and tasks, and they need to be easy to toggle for user to frequently usage.

    After a few trials(top nav-bar, toggle button…), we adopt the tab-bar style Manu, which can provides the ability to quickly switch between different sections. First two tabs are for the two main functions. Then integrate all user related function into third user tab and online point redemption in the last tab.

    #2: Layout

    This version of UI design uses the concept of go station as the mission board and each battery in it represents a mission to add the elements of gamification, but it’s disturbing when using in a small device.

    Therefore the final version use more clear layout for usability.

    Final solutions

    Localized Battery station search

    User's current locations are used to search go-stations that are within 1 km range, so user can see the nearest stations without typing address. User can also search for their route to get go-stations along the way. Bettery inventory are indicated with different color of icons. Green icons indicate sufficient supply, while white icons mean not enough batteries available.

    Besides, user can get 5 points when check-in to the Battery station.

    Mission to complete

    We use user's current location to search stations that are within 5 km range, so user can see the nearest stations without typing address. Bettery inventory are indicated with different color of icons. Green icons indicate sufficient supply, while white icons mean not enough batteries available.

    Besides, user can get 5 points when check-in to the Battery station.

    Badges & Leader Board

    User can see all the points and badges collected. Also the leaderboards for fridens and all users.

    On-Line Points Shop

    User can redeem their points for Gogoro Official Products. Products can be pick up at retail stores and maintenance centers.
